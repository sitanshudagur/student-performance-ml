# student-performance-ml
This project predicts student marks based on factors like study hours, sleep, and attendance.  The goal was to understand how basic features affect performance and to apply Linear Regression in a practical scenario.
## 📊 Dataset
The dataset includes:

- StudyHours → Number of hours studied per day  
- SleepHours → Average sleep per day  
- Attendance → Attendance percentage  
- Marks → Final marks (target)
